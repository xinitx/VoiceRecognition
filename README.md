
###### 文件配置

```

|----------server.js项目主入口文件
|----------routes路由文件夹(里面放要访问的接口地址)
		|----------users.js
		|----------index.js登录和注册功能
|----------db各种连接配置
	|----------db.js连接数据库
	|----------credentials.js创建邮箱证书
|----------server连接数据库操作和路由配置
	|----------register.js用户注册模块
	|----------login.js用户登录模块
|----------dao文件夹(数据库增删改查)
	|----------bcrypt.js对密码进行加密
	|----------dbserver.js数据库服务
	|----------email.js发送邮件服务
	|----------jwt.js token服务
|----------models文件夹
	|----------Article.js文章表
	|----------Comment.js评论表
	|----------Follow.js关注表
	|----------Message.js一对一消息表
	|----------User.js用户表

```


```
开启后端服务器：npm run server
开启前端服务器：npm run serve
```

###### 接口

```
发送验证码到登录邮箱
登录
查询user
更新资料
修改昵称
修改消息数
关注通知
是否为好友（即关注/被关注）
好友聊天
好友即时消息
发帖
删帖
回复评论
删除评论
```
